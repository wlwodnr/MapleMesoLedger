# 🎮 Game Ledger Pro (메이플 재화 관리 대시보드)

웹 브라우저에서 JSON 파일을 직접 읽고 쓰며 게임 재화(메소)와 아이템(조각)을 효율적으로 기록하고 시각화 분석을 할 수 있는 반응형 대시보드 토이 프로젝트입니다.

## 🚀 주요 기능
- **File System Access API 기반의 로컬 파일 직접 수정**: 별도의 백엔드 서버 없이도 사용자가 선택한 로컬 JSON 파일에 실시간으로 기록이 동적 반영 및 자동 저장됩니다.
- **이중 Y축(Dual Y-Axis) 기반의 시각화**: 수량 단위가 다른 '메소(수천만 단위)'와 '조각(한 자릿수 단위)'을 한 차트 안에서 직관적으로 비교 분석할 수 있도록 Chart.js를 커스텀 구현했습니다.
- **실시간 데이터 통합 토글 (Vanishing Bar)**: '조각값 계산' 토글을 활성화하면 조각 수량 막대가 사라지며, 설정한 시세에 맞춰 메소 막대에 가치가 통합되는 시뮬레이션 기능을 제공합니다.
- **탑다운 방식의 기간별 분석**: 1주일(1일 단위), 1개월(3일 단위), 3개월(6일 단위)로 데이터를 유연하게 버킷팅(Bucketizing)하여 자산 성장 트렌드를 추적합니다.
- **데이터 내보내기**: 처음 사용하는 유저나 백업이 필요한 유저를 위해 현재까지의 기록을 `메이플_가계부.json` 파일로 즉시 다운로드하는 기능을 지원합니다.

## 🛠️ 기술 스택
- **Frontend**: HTML5, CSS3, JavaScript (Vanilla ES6+)
- **Library**: [Chart.js](https://www.chartjs.org/) (v4.x via CDN)
